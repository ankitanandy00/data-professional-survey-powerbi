# data-professional-survey-powerbi
Beginner Power BI dashboard analyzing a global data professional survey to uncover salary trends, skill demand, and job satisfaction insights.

# Project Overview
This project presents an interactive Power BI dashboard built to analyze survey responses from data professionals across different countries. The dashboard highlights trends related to salary distribution, job roles, programming language preferences, career difficulty, and job satisfaction.

# Business Objective

The objective of this project is to:

1. Understand key trends in data-related careers
2. Analyze salary variations across job roles
3. Identify in-demand programming languages
4. Evaluate job satisfaction and work-life balance
5. Present insights in a simple and recruiter-friendly dashboard

This analysis can be useful for aspiring data professionals, recruiters, and organizations seeking insights into the data job market.


# Tools & Technologies Used
1. Power BI – Dashboard creation & visualization
2. Power Query – Data cleaning and transformation
3. Excel – Source data format

# Key Questions Answered

This dashboard answers the following questions:

1. Which countries have the highest number of data professionals?
2. What is the average salary across different data job roles?
3. Which programming languages are most preferred in data careers?
4. How difficult do professionals feel it is to break into the data field?
5. How satisfied are professionals with:
6. Work-life balance?
7. Salary?

# Project Process (End-to-End)

## Data Import
Loaded raw survey data into Power BI.

## Data Cleaning & Transformation
- Removed null and inconsistent values.
- Standardized job titles and categorical responses.

## Prepared data using Power Query for analysis.
- Data Modeling
- Created basic calculated measures for:
- Average salary

- Designed an intuitive and clean layout.
- Used appropriate visuals such as:
- Bar charts
- Donut charts

- Gauge visuals

Applied consistent color themes for readability.
Insight Generation
Analyzed visual patterns to extract meaningful career and market insights.

# Key Insights
1. Data Scientists have the highest average salary among surveyed roles.
2. Python is the most preferred programming language across data roles.
3. The majority of respondents feel that breaking into data is moderately difficult.
4. Work-life balance satisfaction is higher compared to salary satisfaction.
5. Most survey responses come from the United States and India.
